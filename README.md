# GRAI App — Grinder Artificial Index

Отдельное Vite-приложение для страницы GRAI (Grinder Artificial Index).

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно на http://localhost:3001

## Сборка

```bash
npm run build
```

## Переменные окружения

- `VITE_MAIN_APP_URL` — URL основного приложения GrindURUS для ссылки «← GrindURUS» в header. По умолчанию `/` (ссылка скрыта).
